## Welcome

This is the official community hub for **Positronic (ASF)** — the first Layer-1 blockchain where 4 neural AI models validate every transaction.

Whether you're a developer, validator, gamer, or just curious — you're in the right place.

## Get Started in 3 Steps

```
Step 1:  Create a wallet     → positronic-ai.network/wallet/
Step 2:  Get free test ASF   → positronic-ai.network/faucet/
Step 3:  Explore the chain   → positronic-ai.network/explorer/
```

## How to Participate

### As a User
- Create a wallet and get testnet ASF from the faucet
- Send transactions, check balances, explore blocks
- Report bugs and earn from the Bug Bounty program

### As a Validator
- Stake 32 ASF to become a validator
- Help secure the network and earn block rewards
- Run a node on your machine or a VPS

### As a Developer
- 213 JSON-RPC methods available
- Python SDK, TypeScript SDK, Telegram Mini App SDK
- Game Bridge SDK for play-to-mine integration
- Full API docs at [positronic-ai.network/docs/](https://positronic-ai.network/docs/)

### As a Gamer
- Play-to-Mine: earn ASF through gaming
- Game Bridge SDK connects any game to the blockchain
- 20% of total supply (200M ASF) allocated to gaming rewards
